Descripción: Aplicación de notas estilo Google Keep.   
Desarrollada utilizando Node.js, MongoDB y Express.  
Cada nota está compuesta de:   
- título  
- contenido  
- autor  
- fecha de creación  
- fecha de última edición  

### ENDPOINTS:  
  
* GET /api/notes  
* POST /api/notes  
  
* GET /api/notes/:id  
* PUT /api/notes/:id  
* DELETE /api/notes/:id
